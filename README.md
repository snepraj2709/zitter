# Zitter App 
A socia media app with the following features

# Features
1. User Feed
2. Create a Post
3. Like a post
4. User Profile
5. Add Bio
6. Add Portfolio URL
7. Follow/Unfollow users
8. Explore Feed
9. Bookmark Post
10. Authentication (using localStorage)
11. Sign-up 
12. Login 
13. Logout
